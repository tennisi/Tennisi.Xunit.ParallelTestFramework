# RetryTheoryAttribute class

Represents a custom xUnit attribute to retry a theory test a specified number of times.

```csharp
[AttributeUsage(AttributeTargets.Method)]
public sealed class RetryTheoryAttribute : TheoryAttribute
```

## Public Members

| name | description |
| --- | --- |
| [RetryTheoryAttribute](RetryTheoryAttribute/RetryTheoryAttribute.md)(…) | Initializes a new instance of the [`RetryTheoryAttribute`](./RetryTheoryAttribute.md) class with the specified retry count. |
| [RetryCount](RetryTheoryAttribute/RetryCount.md) { get; } | Gets the number of retry attempts for the test. |

## Remarks

This attribute is useful for theory tests that may occasionally fail due to transient issues. When used, it will retry the test up to the specified [`RetryCount`](./RetryTheoryAttribute/RetryCount.md) value for each set of inputs.

## Examples

```csharp
[RetryTheory(retryCount: 3)]
[InlineData(1)]
[InlineData(2)]
public void TestMethod(int input)
{
    // Test code here
}
```
