# ParallelTestFramework class

A custom implementation of the xUnit test framework designed to maximize parallelism.

```csharp
public class ParallelTestFramework : XunitTestFramework
```

## Remarks

This framework enables full parallel execution of all `Fact` and `Theory` tests, significantly improving execution speed compared to the default xUnit test framework.

## Examples

To use this framework, update your test project's assembly attribute:

```csharp
[assembly: Xunit.TestFramework("Tennisi.Xunit.v2.ParallelTestFramework", "Tennisi.Xunit.v2.ParallelTestFramework")]
```

Additionally, take advantage of the `ParallelTag` instrument to ensure unique tagging in parallelized tests.
