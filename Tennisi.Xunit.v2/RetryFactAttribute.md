# RetryFactAttribute class

Represents a custom xUnit attribute to retry a test a specified number of times.

```csharp
[AttributeUsage(AttributeTargets.Method)]
public sealed class RetryFactAttribute : FactAttribute
```

## Public Members

| name | description |
| --- | --- |
| [RetryFactAttribute](RetryFactAttribute/RetryFactAttribute.md)(…) | Initializes a new instance of the [`RetryFactAttribute`](./RetryFactAttribute.md) class with the specified retry count. |
| [RetryCount](RetryFactAttribute/RetryCount.md) { get; } | Gets the number of retry attempts for the test. |

## Remarks

This attribute is useful for tests that may occasionally fail due to transient issues. When used, it will retry the test up to the specified [`RetryCount`](./RetryFactAttribute/RetryCount.md) value.

## Examples

```csharp
[RetryFact(retryCount: 3)]
public void TestMethod()
{
    // Test code here
}
```
